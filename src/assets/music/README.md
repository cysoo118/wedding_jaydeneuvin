# Background music

Drop your soundtrack here as `soundtrack.mp3` (the name referenced in
`src/app/wedding-data.ts` → `music.src`).

- Any browser-playable format works: `.mp3`, `.m4a`/`.aac`, `.ogg`.
  If you use a different name or format, update `music.src` to match.
- Keep it reasonably small (a few MB) so the page loads quickly.
- It loops automatically. To turn the music button off entirely, set
  `music.src` to `''` in `wedding-data.ts`.
